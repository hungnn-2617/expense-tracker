Agentic Coding
Mục tiêu:
- Làm quen với việc sử dụng Agent Mode của GitHub Copilot
- Làm quen với một flow SDD cơ bản, thông quan việc sử dụng SpecKit
- Triển khai được một flow SDD để gen code cho một project nhỏ

Cách thức triển khai:
- Làm theo cá nhân. Mỗi người chủ động làm vào thời gian rảnh rỗi 
- Viết document đầy đủ về sản phẩm, chụp ảnh screenshot miêu tả về sản phẩm 
- Submit codes lên repo trên Github cá nhân của công ty, đặt ở chế độ public. Sau đó submit lại đường dẫn đến repo vào dưới mỗi bài tập mà mình lựa chọn sau đây.
Yêu cầu
- Lựa chọn 1 trong 10 đề bài dưới đây 
- Sử dụng SpecKit, hoặc một công cụ tương tự để setup các flow SDD. Có thể tự customize lại flow có sẵn, chỉnh sửa hoặc tạo mới các commands,  templates. 
- Công nghệ: Thoải mái lựa chọn, tuy nhiên nếu được hãy dùng Next.js (cả BE lẫn FE), Typescript, Supabase, và sẽ rất tốt nếu có thể deploy lên CloudFlare Worker hoặc Vercel. 
- MCP: Setup để sử dụng ít nhất 1 MCP Server trong quá trình phát triển, ví dụ như Context7.

Expense Tracker – Ứng dụng quản lý chi tiêu cá nhân
